文老师带你像程序员一样发布个人网站

大家好，我是文科班的编程老师。
对，文科生都开始学编程了，你还有什么理由不学习。

实际问题

昨天，文科班的班长想建立一个班级网站来分享学习笔记。
作为文科班的编程老师，今天我来带大家快速且免费地搭建一个人人可以访问到的网站。

从下载安装开始

文科班的教程当然要像宜家的说明书一样，简单易懂。

如图进入Hexo官网，按照教程下载我们所需要的3个软件：

  • Git
  • Node.js
  • Hexo

文老师假设大家使用Windows 10，安装NodeJs时需要选择64-bit的版本安装。

假装高级程序员，使用github

虽然我们是文科生，我们也要专业地编程。

我们将使用免费的服务器来托管我们的网站。如图所示我们需要在程序员聚集地GitHub上注册一个账户。

然后登陆，如图点击新建按钮进入项目页面。

如图所示，使用username.github.io作为项目名称。但需要把username替换成注册时的用户名称。

文科生善于利用工具

作为文科生一定要善于利用呆萌理科生的一些工具。我们将要使用Hexo来生成网站框架。

大家进入任意一个个人的文件夹，按着Shift键并点击鼠标右键，从菜单中选择＂在此处打开命令行窗口＂。在命令行窗口中输入hexo init website && cd website && hexo server。耐心等待一会儿，当终端窗口不再更新后，网站就成功地在本地运行了。此时可以在浏览地址栏中输入localhost:4000来预览网站。

妈妈再也不用担心我的排版了

Hexo工具默认使用markdown来编写新页面。markdown是一种简单的标记语言，让我们可以专注内容创作而忽略排版问题。只需要花5分钟就可以入门，一个简单的教程传送门。

此时大家可以修改website/source/_posts/hello-word.md文件，来创作我们的第一个页面。完成编辑之后刷新页面，就可以看到成果了。

大家也可以在website文件夹下打开新的命令行并输入hexo new post post_name来新建一个页面。post_name即新文件的名称可以是任何语言的文本，但是建议使用英文命名来避免编码问题。
